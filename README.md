<h1>UpTickets</h1>


<p>
  Uma empresa ficitia voltada para venda de ingressos e eventos de todos os tipos, como: shows, eventos gastronomicos, encontros de fãs e idolos, e etc... Projetos feito com os meus colegas de classe no Senac Santana. O intuito desse projeto é apenas para estudos
